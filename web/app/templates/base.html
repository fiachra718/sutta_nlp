NER Utilities

Jump into annotation tools, browse verses, or explore docs.

Annotate

  • Predict – Paste arbitrary text and run the NER model.
  • Random Verse – Pulls a random verse and opens it in Predict.

Verses

  • Browse verses – Filter by Nikaya, book number, vagga, or verse number and send to Predict.
  • Save to verse – Use Predict to push edits back into ati_suttas.ner_verse_spans.

Docs

  • View training doc – Replace ID to inspect stored training samples.
  • Training doc view – Detailed view with spans table and copyable JSON.

Graph Views

  • ATI study pairs – Ranked discourse pairs with relatedness graph context.
  • ATI study pairs (dense) – Example with higher limit and stricter cosine threshold.
  • Sutta person rank – Suttas ranked by PERSON-network connectivity.
  • Top connected verses – Most connected verses in the mention graph.
  • Community graph example – Person community view centered on Buddha.
  • Sutta PERSON graph example – Focused PERSON network for one sutta.
